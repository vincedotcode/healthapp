## Installation

To run the project locally, follow these steps:

1. **Clone the repository:**

   ```sh
   git clone https://github.com/tusharag6/homestead.git
   cd homestead
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Run the application:**

   ```sh
   npm start
   ```
